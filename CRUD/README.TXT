
Create - Classe que faz a criação dos registros, com base nas informações providas pelo usuário;<br>
Read - Lê algum registro, com base nas informações providas pelo usuário;<br>
Delete - Deleta algum registro, com base nas informações providas pelo usuário;<br>
Update - Atualiza algum registro, com base nas informações providas pelo usuário;<br>
Aux_UPDT - classe com método estático, para fazer a chamada do update (necessária por conta do processo multilinguagem, da biblitoeca Jpype)
